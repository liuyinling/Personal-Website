Algorithms and programming
Public: 25 students in first year
Hourly volume: 84h TD
Responsible for teaching :  Vincent CHEUTET
Description : the objectives of this course were to teach students the principles of object-oriented programming in Java ( Java Programming Environment, Loops and Numerical Sequences, Definition and Calling of Simple Methods - 1 and 2, Arrays and Methods, Characters and Strings, DNA Manipulation ), to upgrade and deepen the knowledge of students in programming through mini-projects ( Game of Life, Langton Ant, Naval Battle, Pipopipette, Minesweeper, Board Game, Matrices ) 
 Role: TP interventions

LEGO EV3 programming
Public : 25 students in third year
Hourly volume : 18h TD
Person in charge of the teaching : Richard MOREAU
Description : the objectives of this course were to make the students understand the JAVA programming adapted to the LEGO EV3, the handling of the EV3 robot and the initialization to the mechatronics, the recovery of the sensors information, the implementation of a multi-task program, and the analysis of the robot performances.
Role: CM and TP interventions
